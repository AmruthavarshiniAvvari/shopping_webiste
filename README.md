# 🛒 Shopping Web Application

## Overview

A full-stack Shopping Web Application built using React.js and Spring Boot. The platform enables users to browse products, search for items, view detailed product information, add products to a shopping cart, and purchase products seamlessly.

The application also provides product management capabilities, allowing administrators to add, update, and manage products efficiently through a secure backend system.

---

## Features

### User Features

* Browse products on the homepage
* View detailed product information
* Search products by name or keywords
* Add products to the shopping cart
* Manage cart items
* Smooth and responsive user interface
* Product image display with pricing and details

### Product Management

* Add new products
* Edit existing products
* Update product information
* Manage product inventory and pricing

### Search Functionality

* Fast product search
* Real-time product filtering
* Improved product discovery experience

## Getting Started

### Clone the Repository

```bash
git clone <repository-url>
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Database Setup

1. Create a MySQL database.
2. Update database credentials in `application.properties`.
3. Run the application.
